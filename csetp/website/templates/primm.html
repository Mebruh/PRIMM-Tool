How Does This Tool Work?

Understanding the PRIMM Learning Methodology

This tool uses the PRIMM teaching methodology to help you build a deep understanding of SQL.

PRIMM stands for: Predict, Run, Investigate, Modify, Make.

Predict and Run

You'll see an SQL statement and need to predict what it will do before running it. Choose from multiple-choice options based on what you think the query will return, then click the RUN button to execute the query and see the actual results. This helps you develop intuition about how SQL works.

Investigate

Answer questions about the SQL statement from the Predict and Run section. These questions help you understand the key concepts like why the query works the way it does, what each clause means, and how the database processes the statement. This deepens your comprehension of SQL syntax and logic.

Modify

You'll receive a task asking you to find specific information in the database, along with a partially correct SQL statement. Your job is to modify the query to make it fully correct and retrieve the requested data. This helps you practice adapting existing SQL code to new requirements.

Make

The final challenge: you'll be given a task and must write your own SQL query from scratch. This is where you apply everything you've learned to create a complete, working query independently. You'll receive helpful hints if your query isn't quite right.
